========================================================
====================[INSTALLATION]======================
========================================================
install nodejs 		@ 	https://nodejs.org/en/
move this folder to root of your drive ex: "C:/" in windows
if windows
	- 32bit = extract nircmd32
	- 64bit = extract nircmd64
	- run nircmd.exe
	- install to root @ C:/
	- run install.bat
	- run run.vbs
else
	- run install.sh
========================================================
=======================[ NOTES ]========================
========================================================
To see the rat on all devices go to your ipv4 address in your browser followed by the port.
	- example url: http://192.168.1.3:8002

To open the rat again run "background.sh" on linux or run "run.bat" on windows.
